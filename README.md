# Java Exception Practice

This project demonstrates basic **exception handling** and **custom exceptions** in Java.

## Examples

### Exercise
- Handles `ArithmeticException`
- Prevents division by zero from crashing the program

### Exercise2
- Creates a custom `InvalidAgeException`
- Validates user age
- Uses `throw`, `throws`, and `try-catch`

## Concepts
- `try-catch`
- `throw` and `throws`
- Custom exceptions
- Exception handling

## Purpose
- Practice Java exception handling
- Learn how to create and use custom exceptions

## Run
Run `Exercise` and `Exercise2` separately.
